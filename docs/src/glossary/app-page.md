# Application Pages

The following are terms used to name pages in the application

## Landing Page
The first page the user is shown before entering the site

![Landing Page Image](/login/login-landing.png)

## Sign In Page
The page with a button for Google's Sign In

![Sign In Page Image](/login/login-signin.png)

## Inbox

The page containing the list of all active and pending requests a user is handling

![Inbox Page Image](/inbox/inbox-empty.png)

## Requests Page

The page containing the list of all pages in a request

![Requests Page Image](/requests/requests-empty.png)

## Request Details Page

The page containing the details of an individual request

![Request Details Page](/requests/request-details-page.png)

## Configuration

The page containing the list of request types and the form to create new request types

![Configuration Page Image](/configuration/configuration-default.png)

## Edit Request Type Page

The page containing the form to edit or delete and existing request type

![Edit Request Type Page](/configuration/edit-request-type-page.png)

## Admin Page

The page exclusive to admins

![Admin Page Image](/admin/admin-statistics.png)
